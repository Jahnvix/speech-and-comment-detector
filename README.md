# 🔥 AI Fake News & Hate Speech Detector

A smart web-based AI application that detects:

* 📰 Fake News
* 🚨 Hate Speech Comments

Built using **Machine Learning + Flask + Modern UI**

---

## 🚀 Features

* 🔍 Fake News Detection using ML model
* 🚨 Hate Speech Detection with hybrid logic (ML + keyword filtering)
* 🎤 Voice Input (Speech Recognition)
* 💡 Safe Phrase Handling (custom file-based system)
* 📊 Confidence Score display
* 🎨 Premium Glassmorphism UI
* 📱 Fully Responsive Design
* ⚡ Real-time prediction

---

## 🧠 Tech Stack

* Python
* Flask
* Scikit-learn
* Pandas
* HTML, CSS, JavaScript

---

## 📂 Project Structure

```
├── app_web.py
├── Fake.csv
├── True.csv
├── hate_speech.csv
├── safe_phrases.txt
├── requirements.txt
├── runtime.txt
├── Procfile
└── templates/
    └── index.html
```

---

## ⚙️ How It Works

1. User enters text or uses voice input 🎤
2. Text is cleaned and processed
3. Model predicts:

   * Fake / Real news
   * Hate / Safe comment
4. Confidence score is displayed

---

## 💻 Run Locally

```bash
pip install -r requirements.txt
python app_web.py
```

Then open:

```
http://127.0.0.1:5000
```
